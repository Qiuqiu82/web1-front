## 📋 后端接口配置

### API 基础地址
```javascript
// src/config/config.js
baseUrl: 'http://localhost:8080/springboot0le12/'
```

### 已对接的接口

#### 1. 登录接口
```
GET /{tableName}/login
参数：username, password, tableName
返回：token, 用户信息
```

#### 2. 注册接口
```
POST /{tableName}/register
参数：用户注册信息（账号、密码、姓名等）
返回：注册结果
```

## 🐛 注意事项

1. **后端服务必须启动**
   - 确保后端在 8080 端口运行
   - 数据库连接正常

2. **跨域问题**
   - 如果遇到跨域，需要后端配置 CORS
   - 或在 vue.config.js 中配置代理

3. **Token 过期**
   - Token 过期会自动跳转到登录页
   - 需要重新登录获取新 Token

4. **文件上传**
   - 上传接口：`file/upload`
   - 返回的路径需要拼接 baseUrl 才能访问
