## 📝 技术要点

### 接口调用规范
```javascript
// GET 请求
this.$http.get('接口路径', {params: {参数}}).then(res => {
  if (res.data.code == 0) {
    // 处理数据
  }
})

// POST 请求
this.$http.post('接口路径', 数据对象).then(res => {
  if (res.data.code === 0) {
    // 处理成功
  }
})
```

### 图片路径处理
```javascript
// 判断是否是完整URL
preHttp(str) {
  return str && str.substr(0,4)=='http';
}

// 使用
v-if="preHttp(item.picture)" :src="item.picture"
v-else :src="baseUrl + item.picture"
```

### 路由跳转
```javascript
// 跳转到详情页
this.$router.push({
  path: '/index/remaixifuDetail', 
  query: {detailObj: JSON.stringify(item)}
})

// 跳转到列表页
this.$router.push({path: '/index/remaixifu'})
```
